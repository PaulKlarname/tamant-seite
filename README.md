# TAMANT – Website

Offizielle Website der TAMANT KI-Beratungsagentur.
Gehostet über **GitHub Pages** unter [tamant.de](https://tamant.de).

## Inhalt

| Datei / Ordner | Beschreibung |
|---|---|
| `index.html` | Hauptseite |
| `impressum.html` | Impressum & Datenschutzhinweise |
| `style.css` | Stylesheet |
| `script.js` | JavaScript |
| `Multimedia/` | Bilder und Videos |
| `ALT/` | Archiv / nicht mehr aktive Dateien |

## Lokale Entwicklung

Die Website ist reines HTML/CSS/JS ohne Build-Schritt.
Zur lokalen Vorschau genügt es, `index.html` im Browser zu öffnen oder
einen einfachen lokalen Server zu starten (z. B. über die VS Code
Erweiterung **Live Server**).

## Deployment

Änderungen werden automatisch über GitHub Pages veröffentlicht, sobald
sie in den `main`-Branch gepusht werden.
